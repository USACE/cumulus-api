cumulus_packager.utils.decompress(src: str, dst: str = '/tmp', recursive: bool = False)[source]

Decompress gzip, tar, tar gzip, or zip file

Destination as a temporary directory best used because this methods does not clean files/directories

Parameters

  • src (str) – input FQPN to compressed file

  • dst (str, optional) – FQP to output directory, by default “/tmp”

  • recursive (bool, optional) – recursive decompress is gzip, tar, tar gzip, or zip file, by default False

Returns

FQP as a directory or single file if not a tar

Return type

str

cumulus_packager.utils.file_extension(file: str, preffix: str = '', suffix='.tif', maxsplit=- 1)[source]

Replace file extension with suffix with allowable extensions

Parameters

  • file (str) – input file name or FQPN; don’t use preffix with FQPN

  • preffix (str, optional) – add preffix to input string, by default “”

  • suffix (str, optional) – replace extension with suffix, by default “.tif”

  • maxsplit (int, optional) – for multiple “.ext”; e.g. .tar.gz, by default -1

Returns

filename or FQPN with new extension, preffix, or both

Return type

str
